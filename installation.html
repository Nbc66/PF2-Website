Requirements

To install Pre-Fortress 2, you must have:

  • Source SDK Base 2013 Multiplayer
  • 7-Zip

Downloads

Google Drive: https://drive.google.com/file/d/1QyFZqSWab2BRFHeRhf1ta7qDLVO5WxUa

Mega: https://mega.nz/file/9CxTXKyT#rqFGOpbJykJqFsBsxUeWaimJF82ygj-BbulEn2Yf8vI

Yandex: https://disk.yandex.ru/d/yegcFzbO1h9Duw


Installation

After downloading the archive from one of the links above, navigate to the sourcemods folder and extract the pf2 folder inside the 7z archive there.
The sourcemods folder can usually be found in these locations:

  • Windows - C:/Program Files (x86)/Steam/steamapps/sourcemods/
  • Linux - ~/.steam/steam/sourcemods/
  • Mac - /Users/USERNAME/Library/Application Support/Steam/steamapps/sourcemods/
